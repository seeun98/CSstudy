### 2.HTTP 1.0 vs HTTP 1.1
```
- HTTP 1.0 한 연결당 하나의 요청만 처리함 . RTT 증가
- HTTP 1.1 매번 TCP 연결을 하는 것이 아니라 한번 TCP 초기화 이후 여러 개의 파일 송수신
   
은진님) 1.0은 한번 연결하면 끊김. 3개의 파일을 연결하려면 계속 연결, rtt 증가
        1.1에서는 단점 보완, 한번 연결하여 쭉 연결함. rtt 줄임
```
